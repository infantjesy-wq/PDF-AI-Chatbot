# 📚 PDF AI Chatbot
An AI-powered chatbot that allows users to upload PDF documents and ask questions about the content using Semantic Search.

## 🚀 Features
* **PDF Text Extraction:** Reads and processes live text from PDFs.
* **AI-Powered Search:** Uses `all-MiniLM-L6-v2` Sentence-Transformers to find the most relevant sentences.
* **Modern Chat UI:** A sleek, user-friendly interface built with Streamlit.
* **Fast & Lightweight:** Optimized for quick responses and low memory usage.

## 🛠️ Tech Stack
* **Frontend:** Streamlit
* **AI Model:** Sentence-Transformers (Hugging Face)
* **Logic:** Python, Cosine Similarity
* **Deployment:** Hugging Face Spaces
